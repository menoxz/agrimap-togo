Le problème

9 familles de données agricoles éparpillées, sans vue d'ensemble. Personne ne voit les vides — les zones où la production existe sans l'écosystème de services.

La solution

Une carte interactive qui croise production et services, révèle les zones blanches et répond à une question : où concentrer le prochain investissement ?

L'impact

Un outil pour les décideurs publics, les partenaires financiers et les acteurs du terrain — afin de cibler les aménagements là où ils créeront le plus de valeur.
